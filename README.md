# TaskTracker — Мини-консольный трекер задач LR23/24 

Простой консольный трекер задач с локальным JSON-хранилищем.

## Возможности
- Добавление, просмотр и удаление задач.
- Расширенный поиск по Title и Description с фильтром по статусу.
- Сохранение последнего фильтра в `config.json`.
- Корзина (soft delete) с восстановлением и полной очисткой (Admin).
- Локальное хранение данных в `Data/tasks.json`.

## Команды
- 1) Добавить задачу
- 2) Показать все активные задачи
- 4) Удалить задачу (отправить в корзину)
- 19) Расширенный поиск/фильтр
- 20) Повторить последний фильтр
- 21) Показать корзину
- 22) Восстановить задачу из корзины
- 23) Очистить корзину (Admin)
