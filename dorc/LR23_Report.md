
## Что сделано
- Добавлен метод `SearchAdvanced(text, status)` в `TaskService` для поиска по:
  - Title
  - Description
- Добавлены поля в `AppConfig`:
  - `LastFilterText`
  - `LastFilterStatus`
- Добавлены команды:
  - 19) Расширенный поиск/фильтр
  - 20) Повтор последнего фильтра
- Последний фильтр сохраняется в `Config/config.json`.

## Как проверить
1. Добавить несколько задач с одинаковыми словами в Title/Description.
2. Через команду 19 ввести текст для фильтрации.
3. Выбрать статус (Any / New / InProgress / Done).
4. Перезапустить приложение.
5. Команда 20 повторяет последний фильтр.
