# Отчёт ЛР24 — Корзина

## Что сделано
- Добавлено поле `IsDeleted` в `TaskItem`.
- Команда Delete (4) отправляет задачу в корзину (soft delete).
- Основной список показывает только активные задачи (`GetAllActive`).
- Добавлены команды:
  - 21) Показать корзину
  - 22) Восстановить из корзины
  - 23) Очистить корзину (только Admin)
- Очистка корзины полностью удаляет задачи и обновляет `tasks.json`.

## Как проверить
1. Добавить 2–3 задачи.
2. Удалить одну задачу — она исчезает из списка.
3. Через команду 21 показать корзину — задача должна быть там.
4. Через команду 22 восстановить задачу — она возвращается в основной список.
5. Через команду 23 очистить корзину (Admin) — задачи удаляются навсегда.
